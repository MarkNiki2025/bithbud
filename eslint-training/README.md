# ESLint Training

This course will guide you through understanding ESLint, exploring the AST, practicing JavaScript and TypeScript, and gradually building your skills by completing exercises and creating ESLint rules.

---

## Installation

Before starting, make sure you are in the **eslint-training** directory:

```bash
cd eslint-training/
```

Then install dependencies:

```bash
npm install
```

It's recommended to create a new branch for your progress:

```bash
git checkout -b eslint-training-<add yor nickname here>
```

Make your first commit:

```bash
git commit --allow-empty -m "init-commit"
```

Push your branch to the remote repository:

```bash
git push --set-upstream origin eslint-training-<add yor nickname here>
```

After that, you can use these commands to save your progress:

```bash
git add .
git commit -m "add your message"
git push
```

---

## How to Complete a Lessons

Each lesson is a test file located in the `lessons/` directory.
Open the lesson file and complete the exercises directly inside it, following the instructions in each test.

## Running Lessons

To run the tests for a specific lesson, use the following commands:

| Lesson | Command            |
| ------ | ------------------ |
| 1      | `npm run lesson-1` |
| 2      | `npm run lesson-2` |
| 3      | `npm run lesson-3` |
| 4      | `npm run lesson-4` |

---

## Who Is This Course For?

### Beginners:

Start with **Lesson 1** and go in order. Each lesson builds your understanding of ESLint and gives hands-on practice.

### Experienced Developers:

You can start with **Lesson 4** to implement ESLint rules. If needed, go back to earlier lessons for hints or refreshers.
