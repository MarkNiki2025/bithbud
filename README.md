# БізБуд 💼

Односторінковий лендинг для консалтингової компанії, що спеціалізується на бізнес-плануванні, бухгалтерських послугах та фінансовій оптимізації.

## 🔗 Структура сторінки

- **Головна** — заголовний екран із заголовком, що плавно з'являється після завантаження
- **Послуги** — три картки з іконками: бізнес-плани, бухгалтерія, фінансова оптимізація
- **Числа** — блок з анімацією появи при скролі
- **Відгуки клієнтів** — карусель імен зі свайпом на мобільних і автоматичною зміною кожні 6 секунд
- **Контакти** — форма зворотного зв'язку та інтерактивна карта Google Maps

## ✨ Можливості

- Плавні анімації появи секцій при скролі (через `IntersectionObserver`)
- Адаптивне бургер-меню для мобільних пристроїв
- Підсвічування активного пункту навігації відповідно до поточної секції
- Карусель відгуків: свайп на тач-екранах, авто-перемикання на десктопі
- Зміна фону хедера при прокрутці
- Вбудована карта Google Maps з кастомним маркером

## 🛠 Технології

- **HTML5** — семантична розмітка
- **CSS3** — адаптивна верстка, переходи й анімації
- **Vanilla JavaScript** — без фреймворків і залежностей
- **Google Maps JavaScript API** — для інтерактивної карти в контактах

## 📂 Файли проєкту

```
bithBud/

├── style.css       # стилі й анімації
├── main.js         # логіка: меню, карусель, скрол-ефекти

```

## 🚀 Запуск локально

Проєкт не потребує збірки — досить відкрити `index.html` у браузері:

```bash
git clone <посилання-на-репозиторій>
cd bithBud
open index.html   # або просто перетягнути файл у браузер
```

Для коректної роботи карти на власному деплої знадобиться власний ключ Google Maps API (замінити в `index.html`, у скрипті підключення карти).

## 📱 Адаптивність

Верстка адаптована під мобільні пристрої: бургер-меню замінює горизонтальну навігацію, а карусель відгуків підтримує керування свайпом.
